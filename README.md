# 💓 HealthGuard AI – Heart Disease Prediction App

HealthGuard AI is a machine learning-powered web application that predicts the risk of heart disease based on patient health parameters.

---

## 🚀 Features
- Predicts heart disease risk using trained ML models
- Interactive web app built with Streamlit
- Real-time prediction based on user input
- Downloadable prediction report (CSV)
- Clean and user-friendly UI

---

## 🧠 Machine Learning
- Models used:
  - Logistic Regression
  - Random Forest (Best Model ~85% accuracy)
  - XGBoost
- Data preprocessing:
  - Missing value handling
  - Encoding categorical variables
  - Feature scaling
- Feature engineering applied

---

## 📊 Dataset
- UCI Heart Disease Dataset (from Kaggle)
- Includes features like:
  - Age
  - Cholesterol
  - Blood Pressure
  - Heart Rate
  - Chest Pain Type

---

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit
- SHAP (Explainable AI)
